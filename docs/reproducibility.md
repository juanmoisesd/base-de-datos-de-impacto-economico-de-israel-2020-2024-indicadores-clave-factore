## 1. Requirements

- Web browser or data analysis software (R, Python, Excel)
- No special software required for basic data access

## 2. Data Access

```
# Clone repository
git clone https://github.com/juanmoisesd/base-de-datos-de-impacto-economico-de-israel-2020-2024-indicadores-clave-factore.git
cd base-de-datos-de-impacto-economico-de-israel-2020-2024-indicadores-clave-factore
```

## 3. Data Format

- Data is provided in standard CSV/HTML format
- UTF-8 encoding
